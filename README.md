# dg-docker-image

Docker image for [Dg programming language](https://pyos.github.io/dg/):

- rzuckerm/dg:`<version>-<tag>`

where:

- `<version>` is the [Dg version](DG_VERSION)
- `<tag>` is the current GitHub tag without the "v"
